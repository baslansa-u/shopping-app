# 🛒 Shopping App (Flutter + Go + Redis)

Simple shopping app using Flutter (BLoC) with Go backend and Redis caching.

---

## 🚀 Tech Stack
- Flutter
- Go (Gin)
- Redis
- JSON (Mock data)

---

## ✨ Features
- Product & Brand list
- Cart system (add/remove)
- Payment summary
- Redis caching
- Static images from server
